## 依赖

* Node.js >= 8

## 安装

```
npm i legoflow-cli -g
```

## 使用

#### 查看命令

```
legoflow -h
```

#### 开发

```shell
# 进入 legoflow 项目
cd test

legoflow dev
```

#### 构建

```shell
# 进入 legoflow 项目
cd test

legoflow build
```
